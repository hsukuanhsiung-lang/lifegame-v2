## 🔍 常见问题排查

### 1. 页面显示空白

**检查点**：
- [ ] 检查 index.html 中的 JS 文件路径是否正确
- [ ] 打开浏览器开发者工具查看 Console 错误
- [ ] 确认所有模块都正确注册 `LifeGame.registerModule()`

**调试代码**：
```javascript
// 在浏览器控制台运行
LifeGame.getModule('tasks');  // 检查模块是否加载
LifeGame.core.Storage.data;   // 检查数据是否加载
```

### 2. 触摸/点击无响应

**检查点**：
- [ ] 检查是否有元素遮罩层 (z-index 问题)
- [ ] 确认事件委托绑定正确
- [ ] 检查是否阻止了默认行为 `e.preventDefault()`

**调试代码**：
```javascript
// 监听所有点击事件
document.addEventListener('click', function(e) {
  console.log('Clicked:', e.target.className);
}, true);
```

### 3. 动画卡顿

**检查点**：
- [ ] 检查是否使用了 `transform` 和 `opacity` (GPU加速)
- [ ] 避免在动画中修改 `width/height/top/left`
- [ ] 检查是否有大量 DOM 操作

**优化建议**：
```css
/* 好的做法 - GPU加速 */
.card {
  transform: translateX(100px);
  opacity: 0.5;
}

/* 避免 - 触发重排 */
.card {
  left: 100px;  /* 会导致重排 */
  width: 200px; /* 会导致重排 */
}
```

### 4. 布局错乱

**检查点**：
- [ ] 检查 viewport meta 标签
- [ ] 使用相对单位 (%, vw, vh) 而非固定 px
- [ ] 检查 flex/grid 布局兼容性

## 🐛 iOS Safari 特有问题

### 问题 1：橡皮筋效果影响拖拽

**现象**：下拉刷新或橡皮筋效果与拖拽冲突

**解决**：
```javascript
document.body.addEventListener('touchmove', function(e) {
  if (e.target.closest('.draggable')) {
    e.preventDefault();
  }
}, { passive: false });
```

### 问题 2：点击延迟 300ms

**现象**：点击后需要等待 300ms 才响应

**解决**：已在 CSS 中设置
```css
touch-action: manipulation;
```

### 问题 3：LocalStorage 大小限制

**现象**：数据保存失败

**检查**：
```javascript
// 检查存储空间
try {
  localStorage.setItem('test', 'x'.repeat(1024 * 1024));
  localStorage.removeItem('test');
  console.log('存储空间充足');
} catch (e) {
  console.log('存储空间不足');
}
```

## 🐛 Android 特有问题

### 问题 1：返回键直接退出

**现象**：点击系统返回键直接退出应用

**解决**：监听返回键事件
```javascript
// 在需要拦截返回键的页面
document.addEventListener('backbutton', function(e) {
  e.preventDefault();
  // 执行返回操作
  LifeGame.emit('nav:back');
}, false);
```

### 问题 2：软键盘弹出布局错乱

**现象**：输入框聚焦时页面被压缩

**解决**：
```javascript
// 监听键盘事件
window.addEventListener('resize', function() {
  if (document.activeElement.tagName === 'INPUT') {
    document.activeElement.scrollIntoView({ behavior: 'smooth' });
  }
});
```

## 📊 性能测试

### 检查列表渲染性能

```javascript
// 测量渲染时间
console.time('render');
module.render(container);
console.timeEnd('render');
// 应该 < 50ms
```

### 检查内存占用

```javascript
// 检查数据大小
var dataSize = JSON.stringify(localStorage).length;
console.log('LocalStorage 使用:', (dataSize / 1024).toFixed(2), 'KB');
// 建议 < 5MB
```

### 检查 FPS

```javascript
// 简单 FPS 检测
var lastTime = performance.now();
var frameCount = 0;

function checkFPS() {
  var now = performance.now();
  frameCount++;
  
  if (now - lastTime >= 1000) {
    console.log('FPS:', frameCount);
    frameCount = 0;
    lastTime = now;
  }
  
  requestAnimationFrame(checkFPS);
}

checkFPS();
```

## 🚀 优化建议

1. **减少 DOM 操作**：批量更新而非逐个更新
2. **使用事件委托**：避免大量事件监听器
3. **图片优化**：如有图片使用懒加载
4. **CSS 优化**：使用 transform 替代位置属性
5. **代码分割**：大模块按需加载

## 📞 调试技巧

### 远程调试 Android

1. 手机开启开发者选项 → USB 调试
2. 电脑 Chrome 打开 `chrome://inspect`
3. 连接手机，授权调试
4. 点击 "Inspect" 查看控制台

### 远程调试 iOS

1. Mac 上打开 Safari → 偏好设置 → 高级 → 显示开发菜单
2. iPhone 设置 → Safari → 高级 → Web 检查器
3. 连接 USB
4. Safari 开发菜单 → 选择你的设备

### 使用 Eruda（移动端控制台）

如需在手机上直接查看控制台：

```html
<!-- 在 index.html 头部添加（仅调试时） -->
```
