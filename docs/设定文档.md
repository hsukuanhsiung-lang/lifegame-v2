## 二、任务系统

### 2.1 任务分类

```
任务
├── 今日任务
│   ├── 突发任务（period: 'today'）
│   └── 到期子任务（来自长期任务，date=今天）
│
├── 长期任务（period: 'long'）
│   └── 子任务（可设date，可标记daily）
│
└── 收集箱（period: 'inbox'）
    └── 无日期备忘，可随时分配
```

### 2.2 子任务属性

| 属性 | 类型 | 说明 |
|------|------|------|
| n | string | 子任务名称 |
| diff | string | 难度 SSS-SS-S-A-B-C-D-E-F |
| done | boolean | 是否完成 |
| date | string | 计划日期 YYYY-MM-DD |
| daily | boolean | 是否每日挑战 |
| subtasks | array | 下级子任务（最多3层） |

### 2.3 子任务规则

| 规则 | 说明 |
|------|------|
| **难度限制** | 子任务难度 ≤ 父级难度（禁止跨级刷经验）|
| **层级限制** | 最多3层嵌套（任务→子任务→子子任务）|
| **自动折叠** | 已完成的子任务自动折叠 |
| **Daily标记** | 设为daily的子任务只在挑战页显示 |

### 2.4 长期任务完成规则

| 规则 | 说明 |
|------|------|
| **完成条件** | 必须完成所有子任务才能标记长期任务完成 |
| **视觉提示** | 未完成时勾选框显示🔒，提示需完成子任务 |
| **取消完成** | 可随时取消完成状态，不受限制 |

### 2.5 排序规则

| 列表 | 排序 |
|------|------|
| 今日任务 | 未完成 ↑ / 已完成 ↓ |
| 长期任务列表 | 未完成 ↑ / 已完成 ↓ |
| 子任务树 | 未完成 ↑ / 已完成 ↓ |

## 三、挑战系统

### 3.1 每日挑战

| 项目 | 规则 |
|------|------|
| 来源 | 只显示 `daily: true` 的长期子任务 |
| 显示位置 | 仅在挑战页面显示 |
| 任务系统 | 今日/长期都不显示daily子任务 |
| 创建方式 | 需选择绑定的长期任务才能创建 |
| 默认等级 | 创建时默认F级（初心）|

### 3.2 等级计算

```
等级 = f(累计完成天数)
```

- 每连续完成一天，累计天数 +1
- 中断后累计天数清零，但等级保留
- 等级只升不降

### 3.3 禁止事项

| 项目 | 规则 |
|------|------|
| 独立性 | 完全独立，不和任务系统关联 |
| 奖励 | 唯一可获得钻石的来源 |
| 机制 | 戒断天数统计，破戒重置 |

## 四、界面交互

### 4.1 今日任务页面

| 功能 | 交互 |
|------|------|
| 日期切换 | 双击"今日"标签显示今天/明天/后天切换器 |
| 收集箱 | 独立标签，可分配到今天/明天/长期 |

### 4.2 长期任务详情页

| 功能 | 交互 |
|------|------|
| 进入方式 | 双击长期任务卡片 |
| 标签页 | 子任务 / 统计 / 设置 |
| 子任务树 | 3层树状显示，自动折叠已完成 |
| 子任务详情 | 双击子任务名称进入 |

### 4.3 子任务详情页

| 功能 | 显示 |
|------|------|
| 路径面包屑 | 根任务 → 父级 → 当前（高亮）|
| 当前卡片 | 难度、名称、日期、完成状态 |
| 下级列表 | 当前子任务的子子任务 |
| 操作按钮 | 编辑、添加下级、删除 |

## 五、架构变化

### 5.1 v1.2.0 → v2.0

| 方面 | v1.2.0 | v2.0 |
|------|--------|------|
| 架构 | 单体HTML | 模块化架构 |
| 命名空间 | 全局函数 | LifeGame命名空间 |
| 模块通信 | 直接调用 | 事件驱动（emit/on）|
| 样式 | 全局CSS | 模块独立CSS |
| 协作 | 单人维护 | 支持多AI协作 |

### 5.2 文件夹结构

```
LifeGame_v2.0/
├── index.html          # 主入口
├── main.js             # 应用引导
├── core/               # 核心层
│   ├── namespace.js    # 模块系统
│   ├── storage.js      # 数据持久化
│   └── renderer.js     # 视图渲染
├── modules/            # 功能模块
│   ├── world-selector/
│   ├── tasks/          # 任务系统
│   ├── challenges/     # 挑战系统
│   ├── timeline/
│   ├── profile/
│   └── guide/
└── docs/               # 文档
    ├── RULES.md        # 规则设定
    └── PROGRESS.md     # 进度报告
```
