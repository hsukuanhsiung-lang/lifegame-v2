## 三、任务系统

### 3.1 任务分类

```
任务
├── 今日任务
│   ├── 突发任务（period: 'today'）
│   └── 到期子任务（来自长期任务，date=今天）
│
├── 长期任务（period: 'long'）
│   └── 子任务（可设date，可标记daily）
│
└── 收集箱（period: 'inbox'）
    └── 无日期备忘，可随时分配
```

### 3.2 子任务属性

| 属性 | 类型 | 说明 |
|------|------|------|
| n | string | 子任务名称 |
| diff | string | 难度 SSS-SS-S-A-B-C-D-E-F |
| done | boolean | 是否完成 |
| date | string | 计划日期 YYYY-MM-DD |
| daily | boolean | 是否每日挑战 |

### 3.3 子任务规则

| 规则 | 说明 |
|------|------|
| **难度限制** | 子任务难度 ≤ 父级难度（禁止跨级刷经验）|
| **层级限制** | 最多3层嵌套（任务→子任务→子子任务）|
| **自动折叠** | 已完成的子任务自动折叠 |
| **Daily标记** | 设为daily的子任务只在挑战页显示 |

### 3.4 长期任务完成规则

| 规则 | 说明 |
|------|------|
| **完成条件** | 必须完成所有子任务才能标记长期任务完成 |
| **视觉提示** | 未完成时勾选框显示🔒，提示需完成子任务 |
| **取消完成** | 可随时取消完成状态，不受限制 |

### 3.3 显示规则

#### 今日任务页面
- 显示：突发任务 + 今天到期的子任务
- **不显示**：标记 `daily: true` 的子任务
- 排序：主线 > 支线 > 过期任务

#### 长期任务页面
- 默认折叠子任务
- **不显示**：标记 `daily: true` 的子任务（即使展开也不显示）
- 显示总经验：该长期任务下所有已完成子任务的经验总和

#### 日期切换器
- 位置：今日任务页面顶部
- 选项：今天 / 明天 / 后天
- 显示对应日期的突发任务 + 到期子任务

### 3.4 过期处理

- **无顺延机制**：今天未完成的任务
- 第二天自动置顶显示在今日任务
- 颜色渐变：越 overdue 颜色越红（紧急度视觉提示）

### 3.5 收集箱机制

| 操作 | 效果 |
|------|------|
| 创建 | 无日期，存入收集箱 |
| 分配日期 | 可设为今天/明天/后天，变成突发任务 |
| 关联长期 | 变成某长期任务的子任务，可设date |

## 四、挑战系统

### 4.1 每日挑战

| 项目 | 规则 |
|------|------|
| 来源 | 只显示 `daily: true` 的长期子任务 |
| 显示位置 | 仅在挑战页面显示 |
| 任务系统 | 今日/长期都不显示daily子任务 |
| 创建方式 | 需选择绑定的长期任务才能创建 |

### 4.2 等级计算

```
等级 = f(累计完成天数)
```

- 每连续完成一天，累计天数 +1
- 中断后累计天数清零，但等级保留
- 等级只升不降

### 4.3 禁止事项

| 项目 | 规则 |
|------|------|
| 独立性 | 完全独立，不和任务系统关联 |
| 奖励 | 唯一可获得钻石的来源 |
| 机制 | 戒断天数统计，破戒重置 |
