## 钻石奖励机制（v2.0新规则）

### 唯一来源：禁止事项
**只有禁止事项可以获得钻石**，任务系统不产出钻石。

### 每日获取规则
| 等级 | 累计天数 | 每日骰子数 | 升级保护卡奖励 |
|------|---------|-----------|---------------|
| F | 1天 | 1颗(1-6钻) | - |
| E | 7天 | 2颗(2-12钻) | 1张 |
| D | 30天 | 3颗(3-18钻) | 2张 |
| C | 180天 | 4颗(4-24钻) | 3张 |
| B | 365天 | 5颗(5-30钻) | 4张 |
| A | 3年(1095天) | 6颗(6-36钻) | 5张 |
| S | 10年(3650天) | 8颗(8-48钻) | 6张 |
| SS | 20年(7300天) | 10颗(10-60钻) | 8张 |
| SSS | 20年+ | 20颗(20-120钻) | 10张 |

### 全勤奖励
- 完成**所有**禁止事项 = 获得当日全部骰子
- 未完成任意一项 = 当日无钻石

### 破戒惩罚
| 等级 | 惩罚 |
|------|------|
| F-A级 | 掉到F级，累计天数清零 |
| S-SSS级 | 保持等级，累计天数清零（不掉级） |

### 保护卡
- **用途**：破戒时消耗1张，抵消惩罚
- **累计**：升级时获得，F→SSS累计可获得210张
- **高等级特权**：SSS级破戒59次仍可保持满级

## 任务系统（v2.0）

### 三标签结构
1. **今日任务** - 突发任务 + 到期子任务
2. **长期任务** - 子任务树状管理，经验解锁完成
3. **收集箱** - 无日期备忘，随时分配

### 子任务系统
- 最多3层嵌套（任务→子任务→孙任务）
- 难度限制：子任务难度 ≤ 父级难度
- 快速日期：编辑时可一键设为今天/明天/后天
- 子任务全貌：双击详情页标签进入完整树状图

### 到期自动显示
- 设置日期的子任务自动出现在今日任务
- 递归遍历所有层级（含孙任务、曾孙任务）
