### 2. 拖拽排序功能（核心修复）

**文件：** `core/drag-sort.js`

**问题：** HTML5 Drag and Drop API 在移动端不支持

**解决方案：**
- 自动检测设备类型
- **桌面端**：保持原有 HTML5 拖拽
- **移动端**：使用 Touch 事件模拟拖拽
  - 长按（400ms）触发拖拽
  - 拖拽时创建克隆元素跟随手指
  - 释放时自动保存排序
  - 支持震动反馈（haptic feedback）

**使用方法不变：**
```javascript
DragSort.init(container, itemSelector, storageKey, onReorder);
```

## 🔧 技术实现细节

### 触摸拖拽原理

```
1. touchstart
   └─ 启动长按计时器（400ms）

2. touchmove
   └─ 移动距离过大则取消计时器

3. touchend (在计时器触发前)
   └─ 取消计时器（普通点击）

3'. touchend (在计时器触发后)
   └─ 正在拖拽中，执行放置逻辑
```

### 长按检测实现

```javascript
addLongPressListener: function(element, handler, duration) {
  duration = duration || 500;
  var timer = null;
  var startX = 0;
  var startY = 0;
  
  element.addEventListener('touchstart', function(e) {
    var touch = e.touches[0];
    startX = touch.clientX;
    startY = touch.clientY;
    
    timer = setTimeout(function() {
      handler(e); // 长按触发
    }, duration);
  });
  
  element.addEventListener('touchmove', function(e) {
    var touch = e.touches[0];
    var moveX = Math.abs(touch.clientX - startX);
    var moveY = Math.abs(touch.clientY - startY);
    
    if (moveX > 10 || moveY > 10) {
      clearTimeout(timer); // 移动过大，取消长按
    }
  });
  
  element.addEventListener('touchend', function() {
    clearTimeout(timer);
  });
}
```
