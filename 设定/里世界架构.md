## 世界观背景

**里世界现状（第2年）**：
- 隐兽入侵已进入第2年，世界快沦陷
- **大型城镇**：还有秩序，但人满为患
- **中型村庄**：部分被占领，部分在抵抗
- **野外**：极度危险，D级隐兽已算BOSS
- **新手村**：偏远破落，只有村长，距离最近城镇不远（路上最多1只E级隐兽）

**Kuan的处境**：
- 出生等级：F级见习冒险者
- 村口隐兽：F级白色（5-10级可挑战）
- 向导Wiz：穿越时这里还没沦陷，现在也不认识路了

## 详细设计

### 1️⃣ 玩家标签

**三级标签内容**：

| 子标签 | 内容 |
|--------|------|
| **等级属性** | 冒险者等级（F-SSS）、5维属性（HP/STR/INT/AGI/自由）、六柱可视化展示 |
| **灵魂形态** | 当前形态（光团/简笔画/像素）、进化进度、解锁条件 |
| **契合度** | 实时契合度数值、历史趋势图、提升记录 |
| **成就墙** | 已解锁徽章、里程碑、特殊称号 |

**注意**：不显示"今日状态"概览（表世界已有），避免重复。

---

### 2️⃣ 世界标签

**核心功能**：
- **当前位置**：玩家所在地图区域（新手村/废墟/森林等）
- **世界地图**：
  - 已探索区域：清晰可见，可查看详情
  - 未探索区域：战争迷雾覆盖
  - 可缩放、可点击区域查看信息
- **区域信息**：
  - 该区域的隐兽等级分布
  - 资源类型（可采集）
  - 危险度评级
  - 距离新手村的路程

**地图分级**：
- 安全区（新手村周边）：F-E级隐兽
- 危险区（野外）：D-C级隐兽
- 沦陷区（禁区）：B级以上，需收复

---

### 3️⃣ 技能标签

**核心功能**：

| 子标签 | 内容 |
|--------|------|
| **技能工坊** | 碎片合成（100碎片=技能）、技能升级（100下级=上级）、技能分解 |
| **碎片仓库** | 各色碎片存储、数量统计 |
| **技能装备** | 已拥有技能列表、装备/卸下操作、技能排序 |
| **隐兽图鉴** | 已击杀隐兽记录、外观描述、弱点分析、掉落记录 |

**隐兽图鉴详情**：
- 每种隐兽的立绘/描述
- 弱点标注（下次战斗对该类型隐兽+伤害）
- 掉落统计（记录后显示掉率）
- 击杀数量

### 4️⃣ 进化标签

**核心功能**：

#### 灵魂进化（与神交易）
- 使用**钻石**与神交易，进化灵魂形态
- 进化路径：
  - 6钻石 → 简笔画形态
  - 18钻石 → 像素小人形态
  - 后续形态待解锁

#### 商城系统

**货币体系**：
```
1龙币 = 100金币 = 10,000银币 = 1,000,000铜币
```

**货币用途**：
- **铜币/银币/金币**：里世界日常消费，玩家/NPC交易
- **龙币**：高价值交易，稀有商品
- **钻石**：只能与神交易（灵魂进化），不能用于普通消费

**动态定价机制**：
- **地区差异**：不同地图卖不同商品
- **供需定价**：
  - 当地盛产/常见属性 → 便宜
  - 当地稀缺/罕见属性 → 贵

**示例**：
- 冰原地区：冰系技能书80铜币（本地产出），火系技能书250铜币（外地运来）
- 火山地区：火系技能书70铜币（本地产出），水系技能书300铜币（珍贵）
- 大城市：商品全，价格高（供需+运输成本）
- 偏远村庄：商品少，价格低（本地特产便宜，外来货贵）

### 6️⃣ 禁区收复标签

**核心设定**：被隐兽占领的村庄、哨站、矿坑、遗迹等，内有战死的前人装备和残留魂体。

**二级标签结构**：

| 子标签 | 内容 |
|--------|------|
| **沦陷区** | 当前被占领的区域列表，显示难度和预估战利品 |
| **收复区** | 已打下来的区域，可搜刮剩余资源 |
| **战利品** | 从前人遗体获得的装备、记录 |
| **防御工事** | 防止隐兽反扑的建设（可选深度玩法） |

#### 详细机制

**沦陷区挑战**：
```
每个禁区显示信息：
- 等级（F/E/D/C/B/A/S...）
- 隐兽数量（1-5只）
- 预估战利品（前人装备等级、碎片数量）
- 收复难度（推荐等级）
- 距离（从新手村出发的路程）

挑战消耗：
- 契合度（灵魂出窍深入险地消耗能量）
- 或时间（1小时现实时间=挑战1个禁区）

挑战结果：
- 成功：区域变为"收复区"，获得前人装备+碎片+剧情
- 失败：损失部分契合度，退回新手村治疗
```

**收复区搜刮**：
```
每个收复区包含：
- 前人装备箱（一次性，搜完就空）
- 残留魂体（低等级隐兽，可挂机刷少量碎片）
- 剧情记录/日记（了解前人故事，非无限）

注意：资源真的会被搜完，不是无限产出
```

**战利品系统（装备）**：
```
装备类型：
- 武器：增加里世界攻击力
- 防具：增加防御力/血量
- 饰品：特殊效果（+暴击率、+碎片掉落率等）

装备特性：
- 有耐久度，战斗会损耗
- 坏了需要修理（消耗铜币）或更换
- 可以强化（消耗碎片+铜币）
- 装备有等级要求（F级装备F级可用，A级装备需要A级冒险者）
```

**防御工事（可选）**：
```
收复后如果不维护，隐兽可能反扑（低概率事件）
可建设：
- 警戒哨：预警隐兽接近
- 灵魂锚点：传送点，快速回新手村
- 防御阵：降低被反扑概率

维护消耗：铜币 + 时间投入
```

## 关键设计原则

1. **非无限资源**：禁区战利品是一次性的，搜完就空，符合"资源枯竭"的世界观
2. **离线收益合理化**：下线挂机解释为"传送到前线或野外击杀无限多的隐兽"，不是凭空产生资源
3. **难度递进**：村口F级→路上E级→野外D级→禁区B级起，符合当前Kuan的实力定位
4. **表里分离**：里世界战斗不影响表世界属性获取，但通过契合度产生联动
5. **货币分层**：铜银金龙用于人间交易，钻石专用于与神交易（灵魂进化）

---

## 与技能系统的关联

- **碎片来源**：击杀隐兽（离线挂机+禁区挑战）
- **装备来源**：禁区搜刮前人遗体
- **技能作用**：仅在里世界战斗生效
- **战斗形式**：
  - 离线挂机：自动获得碎片（解释：在前线击杀低级隐兽）
  - 禁区挑战：手动战斗，获得装备+碎片
