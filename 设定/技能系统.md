## 一、世界观背景

**原本**：里世界居民通过练习和学习获得技能  
**隐兽入侵后**：击杀隐兽可获得技能碎片，收集碎片合成技能

> "隐兽是里世界异变的产物，它们体内凝聚着异化的能量。击败它们，就能提取这些能量，转化为冒险者可用的技能。"
> ——冒险者工会档案

## 二、碎片系统

### 2.1 碎片等级与颜色

| 颜色 | 等级 | 对应隐兽 | 说明 |
|------|------|----------|------|
| ⚪ 白色 | F级 | 普通隐兽 | 最常见的基础碎片 |
| 🔘 灰色 | E级 | 强化隐兽 | 比普通隐兽更强壮 |
| 🟢 绿色 | D级 | 精英隐兽 | 有一定智慧的个体 |
| 🔵 蓝色 | C级 | 首领隐兽 | 统领小群体的头目 |
| 🟣 紫色 | B级 | 区域BOSS | 占领一片区域的强大存在 |
| 🟠 橙色 | A级 | 深渊领主 | 来自深渊的高阶生物 |
| 🟡 金色 | S级 | 传说隐兽 | 百年难遇的稀有存在 |
| 🔴 红色 | SS级 | 灭世级 | 足以毁灭城镇的恐怖存在 |
| ⚪✨ 光白 | SSS级 | 神级 | 传说中的神明造物 |

### 2.2 碎片获取方式

**击杀隐兽掉落**：
- 不同等级隐兽掉落对应颜色碎片
- 每次击杀随机掉落 1-3 个碎片
- 有极低概率越级掉落（如F级隐兽掉灰色碎片）

**下线挂机收益**：
- 关闭APP后灵魂自动狩猎
- 根据契合度和属性获得碎片
- 有灵魂锚点保护机制防止损失

### 2.3 碎片合成规则

**规则一：同色合成**
```
100个同色碎片 → 合成1个同颜色随机技能

示例：
100个白色碎片 → 随机F级技能
100个蓝色碎片 → 随机C级技能
```

**规则二：等级晋升**
```
100个下级技能 → 合成1个高一级技能

示例：
100个白色技能(F) → 1个灰色技能(E)
100个灰色技能(E) → 1个绿色技能(D)
...以此类推
```

**规则三：技能分解**
```
1个技能 → 分解为50个对应颜色碎片（回收一半）

用于：回收不需要的技能，重新合成
```

## 四、技能作用范围

### 4.1 核心原则

> **里世界技能只在里世界生效，不影响表世界属性获取**

**表世界（现实）**：
- 通过时间轨迹、每日挑战、禁止事项、任务系统获得属性
- 与里世界技能系统完全独立
- 表世界努力 → 提升契合度 → 里世界战斗更强

**里世界（游戏）**：
- 通过技能碎片和技能系统变强
- 用于击杀隐兽、探索秘境、推进剧情
- 里世界成果 → 获得碎片、装备、剧情解锁

### 4.2 联动机制（可选）

**单向联动**（已确定）：
```
表世界表现 → 里世界增益
例：连续7天完成每日挑战 → 里世界获得"专注光环"(7天有效期)
```

**反向联动**（待定）：
```
里世界成果 → 表世界奖励
例：击杀A级隐兽 → 获得"勇者证明"(一次性经验加成)
```

## 五、战斗系统

### 5.1 三模式战斗

#### 模式一：下线挂机（主力收益）

**触发条件**：关闭LifeGame APP

**机制**：
- 灵魂自动进入里世界浅层
- 根据契合度和属性自动击杀隐兽
- 有灵魂锚点保护机制

**收益**：
- 稳定获得碎片（主要来源）
- 基础经验值
- 小概率获得装备

**保护机制**：
- 灵魂锚点：下线前设置，防止高级隐兽攻击
- 自动撤退：HP低于20%自动回城
- 背包上限：碎片满了停止战斗

**惊喜事件**（上线时触发）：
- "你回来啦！挂机期间遇到了一只迷路的小隐兽，掉了一个蓝色碎片！"
- "警告：你的灵魂锚点快失效了，差点被C级隐兽袭击！"

#### 模式二：里世界手动（精华内容）

**触发条件**：打开里世界界面主动进入

**机制**：策略卡牌战斗
- 每回合抽3张技能卡
- 选择打出，组合效果
- 可随时暂停

**限制**：
- 每天最多3次"深度探索"
- 消耗契合度
- 适合BOSS战、技能合成、剧情推进

**收益**：
- 高价值BOSS掉落
- 稀有技能获取
- 剧情推进
- 装备获取

#### 模式三：表世界联动（偶尔触发）

**触发条件**：完成特定现实挑战

**示例**：
- 连续7天完成每日挑战 → 解锁"深渊裂隙"
- 健身突破个人记录 → 遭遇"肌肉巨兽"
- 冥想达到30分钟 → 进入"精神世界BOSS战"

**特点**：
- 不强制，奖励丰厚
- 现实努力的"意外惊喜"
- 不是必须完成的任务

## 六、防肝设计

### 6.1 挂机上限

- 最多累积12小时收益
- 超过12小时不再获得新碎片
- 保留已获得的碎片

### 6.2 手动战斗次数

- 每天最多3次"深度探索"
- 用完后可以观战或整理背包
- 不影响挂机收益
